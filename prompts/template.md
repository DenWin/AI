# AI Prompt Template

## Prompt Structure

Use this template as a foundation for creating structured prompts that can be referenced across different AI systems.

### Basic Template Format

```
# [PROMPT TITLE]

## Context
[Describe the context and background information]

## Objective
[Clear statement of what you want the AI to accomplish]

## Instructions
[Step-by-step instructions or guidelines]
1. [First instruction]
2. [Second instruction]
3. [Additional instructions as needed]

## Input Format
[Specify the expected input format]

## Output Format
[Specify the expected output format]

## Examples
### Input:
[Example input]

### Output:
[Example output]

## Constraints
- [List any constraints or limitations]
- [Additional constraints]

## Quality Criteria
- [Define what constitutes a good response]
- [Additional quality measures]
```

### Template Variables

Use these placeholder patterns in your prompts:

- `{CONTEXT}` - For contextual information
- `{INPUT}` - For user input
- `{OUTPUT_FORMAT}` - For desired output format
- `{CONSTRAINTS}` - For specific limitations
- `{EXAMPLES}` - For example inputs/outputs

### Best Practices

1. **Be Specific**: Use clear, unambiguous language
2. **Provide Context**: Give sufficient background information
3. **Set Expectations**: Clearly define desired outcomes
4. **Include Examples**: Show what good output looks like
5. **Define Constraints**: Specify any limitations or requirements
6. **Use Structure**: Organize prompts with clear sections
7. **Test Iteratively**: Refine prompts based on results

### Reusability Guidelines

- Design prompts to be modular and reusable
- Use consistent terminology across prompts
- Reference this template for maintaining structure
- Document any custom modifications or extensions
